Project 1: Khana-Express

This full-stack MERN application allows users to browse various food items, customize orders, add them to the cart, and proceed to checkout. I implemented features like real-time cart updates, dynamic item filtering, and responsive design to enhance user experience. The backend supports secure authentication and stores user data using MongoDB. The app is optimized for both desktop and mobile devices.

Players can reset the game and start a new round with a single click. It is fully responsive and designed to work on both desktop and mobile devices.

  • Technologies: HTML, CSS, JavaScript
  • Features: Game Logic, Winner Detection, Responsive Design
